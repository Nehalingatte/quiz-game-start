# Quiz Game

## 📌 Overview
The **Quiz Game** is an interactive console-based game that allows users to answer multiple-choice questions and get instant feedback on their responses. This project is designed to test and improve general knowledge while making learning fun.

## 🎯 Features
- 📝 Multiple-choice questions
- ⏳ Timed responses (optional)
- ✅ Instant feedback on answers
- 📊 Score tracking
- 🔄 Play again option

## 🛠️ Technologies Used
- Python

## 🚀 How to Run the Project
1. **Clone the Repository**
   ```sh
   git clone https://github.com/Nehalingatte/quiz-game.git
   cd quiz-game
   ```
2. **Run the Script**
   ```sh
   python quiz_game.py
   ```

## 📌 How to Play
1. The game presents a series of multiple-choice questions.
2. Choose the correct option by entering the corresponding number.
3. Receive instant feedback on your choice.
4. At the end, view your total score.
